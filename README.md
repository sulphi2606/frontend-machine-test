# Frontend Machine Test

## Overview

This project is a frontend implementation using Next.js, TypeScript, and Ant Design. The application includes order management, filtering, personnel assignment, clerk management, and order detail views while following a reusable component-based architecture.

## Tech Stack

* Next.js (App Router)
* TypeScript
* Ant Design
* React

## Features

* Order Management Dashboard
* User Filtering
* Product Filtering
* Tag Filtering
* Order Details View
* Personnel Assignment
* Add Clerk Functionality
* Responsive Design
* Reusable Components

## Project Structure

```text
app/
components/
public/
types/
data/
```

## Getting Started

Install dependencies:

```bash
npm install
```

Run the development server:

```bash
npm run dev
```

Open http://localhost:3000 in your browser.

## Production Build

```bash
npm run build
npm start
```
